## 3. 텍스트 입력 기능 개선

### 실시간 텍스트 미리보기
- textarea 타이핑 시 canvas에 실시간 렌더링
- 편집 중인 element를 canvas 렌더링에서 제외하고 미리보기로 대체
- 텍스트 삭제 시에도 즉시 반영 (`if (inputPos)` 조건으로 수정)

### 텍스트 박스 스크롤 방지
- `autoFocus` 제거 → `focus({ preventScroll: true })`로 교체
- 텍스트 박스 생성 시 화면 스크롤 방지

### 텍스트 박스 색상 고정 버그 수정
- `handleTextClick` hit test를 `el.type === 'text'`만 대상으로 제한
- 원/사각형 위에서 텍스트 도구 클릭 시 해당 도형이 사라지던 문제 해결
- 기존 텍스트 편집 시 `textBgOpacity` 덮어쓰기 제거 (사용자 설정 유지)
- V체크 시 현재 선택 색상이 텍스트에 반영되도록 `newProps.style.copy({ color })` 추가

## 6. 형광펜/사각형/원 텍스트 스냅 기능

### 텍스트 스냅 동작 방식
- **드래그 중**: raw 드래그 rect를 미리보기로 표시
- **마우스 업 시**: 드래그 rect에 걸친 글자들의 실제 bounding box로 확장 (걸친 것 포함)
- 다른 줄 침범 방지: 텍스트 런의 Y 범위로 줄 감지

### 글자 너비 정밀 계산
- 기존 `avgCharW`(평균 너비) → `ctx.measureText()`로 각 글자 실제 너비 측정
- 전체 블록 너비에 맞게 비례 스케일링으로 콤마/공백 등 좁은 문자 오차 해소
- **파일**: `src/tools/next/ShapeTool.ts`, `src/components/viewers/PdfViewer.tsx`

### 화살표 도구 Ctrl 스냅
- 화살표 그릴 때 Ctrl+드래그 시 PDF 텍스트 + 텍스트 박스 + 모든 도형에 스냅
- 시작점/끝점 모두 스냅 적용
- **파일**: `src/tools/next/ShapeTool.ts`

## 8. PDF 렌더링 안정성 개선

### 중복 렌더 에러 수정
- `renderPage()`에서 이전 renderTask를 `cancel()` 후 새 render 시작
- `RenderingCancelledException` 무시 처리
- `willReadFrequently: true` 옵션 추가로 성능 경고 해소

### Invalid page request 에러 수정
- `loadPage()`에서 `pageNum < 1 || pageNum > doc.numPages` 가드 추가
- 새 PDF 로드 시 `pdfProxies` 캐시 초기화
- `useEffect`에서 `currentPage`가 `numPages` 범위 안일 때만 `loadPage` 호출
